This website will allow users to upload images of handwritten text, and the server will return a string of text of what it thinks it says through a handwriting, optical character recognition, convolutional neural network. On Client-side, page will upload images to the server, then output the results of the server’s analysis of the image. By using Tesseract and OpenCV to detect distinct blocks of text, we can output the text for each block separately. Client-side functionality could potentially be expanded by capturing photos from the user’s webcam, or even by saving handwriting drawn on the page.
